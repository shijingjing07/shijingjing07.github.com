C++重载，重写，重定义

1.重载：
重载是在一个类中，函数名一样，参数类型或参数个数不同的一系列函数
需要注意的是，与返回类型无关。

class Test
{
public:
    void show();
    void show(int a);
    void show(double a);
    void show(int a,int b);
    void show(int a,double b);
};

2.重写
我们比较熟悉的另一个词是”覆盖”
用在继承中，子类若没有重新定义父类的虚方法，则子类对象调用该方法时，使用的仍是父类的方法；
否则，子类覆盖了父类的虚方法，调用时使用的是重新定义的方法。

class Test
{
public:
    virtual void show()
    {
        std::cout<<"Test::show()"<<std::endl;
    }
};
class SubTest:public Test
{
public:
    virtual void show()
    {
        std::cout<<"SubTest::show()"<<std::endl;
    }
};
int main()
{
    Test *p = new SubTest;
    p->show();
    return 0;
}

输出结果：
SubTest::show()

3.重定义
我们比较熟悉的另一个词是”隐藏”
子类重新定义父类中有相同名称的非虚函数，则对象调用该非虚方法时，使用的对象本身定义的方法。

class Test
{
public:
        void show()
    {
        std::cout<<"Test::show()"<<std::endl;
    }
};
class SubTest:public Test
{
public:
    void show()
    {
        std::cout<<"SubTest::show()"<<std::endl;
    }
};
int main()
{
    Test *p = new SubTest;
    p->show();
    return 0;
}

输出结果：
Test::show()

关于重写与重定义，如果了解了虚函数表的原理，就很容易理解了，
关于虚函数表，可参考C++虚函数表与虚析构函数
